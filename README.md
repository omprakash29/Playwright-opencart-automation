# Playwright OpenCart Automation

This repository contains a "test automation framework" built using "Playwright (JavaScript)" for the OpenCart application.  
It is designed to support **daily commits**, CI/CD integration, and scalable test execution.

---

🚀 Features
- End-to-end test automation using Playwright
- Page Object Model (POM) structure
- Easy integration with CI/CD pipelines
- Supports multiple browsers (Chromium, Firefox, WebKit)
- Configurable test reports
